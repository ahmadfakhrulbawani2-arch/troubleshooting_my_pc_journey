Everytime I solved a problem on my computer or others, mostly for my projects, I will put it here. This repo is intended to help you guys fix your computer problem the right way, the clearest way possible.

## My General Advice

We all agree that we always encounter different problem every time we use our desktop. It's ok and fine, don't worry too much like I do. For about 10+ years of my life I'm using my parent's PC until my own laptop, here's the steps I learned to solve our computer desktop problem:

1. JUST GOOGLE IT, AI WILL SUMMARIZE LIKE 60-70% of THE ACTUAL SOLUTION
2. Find indexed tech forum, e.g. Stack Overflow, CodeRanch, Github, etc. WARNING: ALWAYS READ THE COMMENT FROM PEOPLE THAT TRY THE SOLUTION
3. If still not solved, find the unindexed tech forum, e.g. Reddit, Discord, etc.
4. Still not solved? Ask question to the forum.
5. Want fast solution? The last resort is to use AI. I'm not recommend you to use AI because they're so suck and they're basically just way more powerful search engine like Google.
